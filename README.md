# FUTURE_DS_02
This project analyzes customer behavior on online retail  data in Power BI. The dashboard shows repeat purchases, customer revenue contribution, retention trends, and regional customer analysis.

Tools Used-Power BI

Online Retail Dataset
KPIs
Total Customers
Total Revenue
Total Orders

Dashboard Features
-Customer purchase frequency analysis
Top customers by revenue
Monthly customer activity trends
Country customer distribution


Key Insights
A small number of customers generated most revenue.
Repeat purchases indicate strong customer retention.
Customer activity increased over time.
Most customers originated from the United Kingdom.


Recommendations
Focus on retaining high-value customers.
Improve loyalty strategies for repeat buyers.
Expand customer acquisition strategies in lower-performing regions.
